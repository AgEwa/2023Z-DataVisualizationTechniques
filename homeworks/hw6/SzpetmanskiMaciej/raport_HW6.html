Opis narzędzia

Seaborn jest jak już wyżej wspominałem paczką do języka Python. Jest to zgrabna oraz efektywna biblioteka, pozwalająca na szybkie tworzenie atrakcyjnych wykresów. Została zbudowana na bazie biblioteki Matplotlib wzbogaconej o dodatkowe typy wykresów.

Seaborn pomaga eksplorować i rozumieć dane użytkownika. Jego funkcje kreślenia działają na ramkach danych i tablicach zawierających całe zestawy danych i wewnętrznie wykonują niezbędne mapowanie semantyczne i agregację statystyczną w celu tworzenia wykresów informacyjnych. Jego deklaratywny interfejs API zorientowany na zestaw danych pozwala skupić się na tym, co oznaczają różne elementy wykresów, zamiast na szczegółach ich rysowania.

Jedną z głównych zalet korzystania z paczki dygraphs jest jej zdolność do produkcji wysokiej jakości wykresów, które są łatwe do interpretacji. Paczka została zaprojektowana z myślą o użytkowniku i zawiera szereg funkcji, które umożliwiają szybkie tworzenie profesjonalnie wyglądających wykresów.

Przykład użycia

import seaborn as sns
import matplotlib.pyplot as plt
import pylab

sns.set_theme(style="whitegrid")

# Load the example diamonds dataset
diamonds = sns.load_dataset("diamonds")

# Draw a scatter plot while assigning point colors and sizes to different
# variables in the dataset
f, ax = plt.subplots(figsize=(6.5, 6.5))
sns.despine(f, left=True, bottom=True)
clarity_ranking = ["I1", "SI2", "SI1", "VS2", "VS1", "VVS2", "VVS1", "IF"]
sns.scatterplot(x="carat", y="price",
                hue="clarity", size="depth",
                palette="ch:r=-.2,d=.3_r",
                hue_order=clarity_ranking,
                sizes=(1, 8), linewidth=0,
                data=diamonds, ax=ax).set(title = 'Gem clarity depending on their price and carat value')
pylab.show()
